## 🚀 C 端功能

### 1. 车险询价

**功能描述**：用户填写车辆信息，实时获取多家保司的智能报价。

**用户场景**：
- 车主需要购买车险
- 想要对比多家保司价格
- 希望获得专业推荐

**输入项**：
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 车牌号码 | 文本 | 是 | 如：沪 A·88888 |
| 车架号 | 文本 | 是 | 17 位 VIN 码 |
| 注册日期 | 日期 | 是 | 车辆首次注册日期 |
| 能源类型 | 选项 | 是 | 燃油车/纯电动/混合动力 |
| 新车购置价 | 数字 | 是 | 单位：元 |
| 承保区域 | 选项 | 是 | 如：上海市 |

**输出项**：
| 字段 | 类型 | 说明 |
|------|------|------|
| 推荐保司 | 文本 | 综合评分最高的保司 |
| 总保费 | 数字 | 交强险 + 商业险 |
| 交强险 | 数字 | 基础保费 |
| 商业险 | 数字 | 商业险保费 |
| 折扣率 | 百分比 | 优惠折扣 |

**业务规则**：
- 车架号必须为 17 位
- 新车购置价必须 > 0
- 承保区域必须选择

---

### 2. 智能推荐

**功能描述**：基于策略引擎和多因子评分，AI 自动推荐最优保险方案。

**推荐维度**：
| 维度 | 权重 | 说明 |
|------|------|------|
| 价格优势 | 40% | 保费竞争力 |
| 服务质量 | 35% | 服务质量评分 |
| 赔付效率 | 25% | 赔付周期和效率 |

**推荐理由**：
- 价格最优
- 服务最好
- 赔付最快
- 综合最优

---

### 3. 方案对比

**功能描述**：多维度对比不同保司的保险方案，帮助用户做出决策。

**对比维度**：
| 维度 | 说明 |
|------|------|
| 保费价格 | 总保费、交强险、商业险 |
| 保司评分 | 综合评分、各维度评分 |
| 保障范围 | 险种覆盖情况 |
| 服务质量 | 服务评分、用户评价 |

### 2. 策略实验室

**功能描述**：提供模拟测试和 A/B 测试能力，验证策略效果。

#### 2.1 模拟测试

**功能描述**：通过情景仿真，提前预估策略调整的影响。

**测试流程**：
1. 选择测试策略
2. 设置测试参数
3. 启动模拟测试
4. 查看测试结果
5. 保存测试结果

**情景参数**：
| 情景 | 赔付率调整 | 保单数调整 | 说明 |
|------|-----------|-----------|------|
| 情景 1 | -4% | -8% | 收紧核保 |
| 情景 2 | -2% | -4% | 轻微收紧 |
| 基准 | 0% | 0% | 当前策略 |
| 情景 4 | +2% | +4% | 轻微放宽 |
| 情景 5 | +4% | +8% | 放宽核保 |

**测试结果**：
| 指标 | 说明 |
|------|------|
| 保费规模 | 总保费收入 |
| 纯风险成本 | 风险成本 |
| 利润率 | 利润率变化 |
| 利润 | 利润变化 |

#### 2.2 A/B 测试

**功能描述**：对比不同策略的实际效果，科学验证策略优劣。

**测试流程**：
1. 设计实验方案
2. 设置实验组和对照组
3. 分配流量
4. 运行实验
5. 分析结果

**实验指标**：
| 指标 | 说明 |
|------|------|
| 成交率 | 成交转化率 |
| 利润率 | 单位经济利润率 |
| 保费规模 | 总保费收入 |
