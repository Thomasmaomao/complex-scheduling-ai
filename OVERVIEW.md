## 🎯 核心方法论

### 1. 业务分群

将业务按维度拆分，独立核算每一块的收支情况。

**维度示例**：
- 地区（上海、北京、广东...）
- 客户类型（新保、续保）
- 渠道类型（直销、代理）
- 产品类型（燃油车、新能源车、豪车）

### 2. UE 建模

单位经济模型：量化收入、成本、风险。

**核心公式**：
```
利润率 = 1 - 赔付率 - 市场费用率 - 固定成本率
利润 = 保费规模 × 利润率
```

### 3. 情景仿真

政策变化的敏感性分析，5 种情景推演。

**情景参数**：
```
情景 1：赔付率 -4%，保单数 -8%   （收紧核保）
情景 2：赔付率 -2%，保单数 -4%   （轻微收紧）
基准：赔付率 0%，保单数 0%       （当前策略）
情景 4：赔付率 +2%，保单数 +4%   （轻微放宽）
情景 5：赔付率 +4%，保单数 +8%   （放宽核保）
```

### 4. 策略优化

基于数据分析找到最优策略，持续迭代改进。

**优化目标**：
- 利润最大化
- 风险可控
- 业务可持续
