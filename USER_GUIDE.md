### 1. 车险询价

#### 步骤 1：进入询价页面

1. 访问系统首页
2. 点击"智能服务"卡片
3. 进入"车险询价示例"页面

**必填信息**：

| 字段 | 示例 | 说明 |

| 注册日期 | 2020-01-01 | 车辆注册日期 |

**操作**：
- 查看方案详情
- 对比不同方案
- 选择心仪方案

### 2. 智能推荐

#### 查看推荐理由

在结果页面，查看 AI 推荐理由：

- 💰 **价格优势**：保费竞争力分析
- ⭐ **服务质量**：服务质量评分
- ⚡ **赔付效率**：赔付周期分析

#### 理解评分

**评分维度**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 价格优势 | 40% | 保费越低分越高 |
| 服务质量 | 35% | 服务评分越高越好 |
| 赔付效率 | 25% | 赔付越快分越高 |

**评分等级**：
- 90-100 分：优秀
- 80-89 分：良好
- 70-79 分：一般
- < 70 分：较差

## 🏢 B 端使用指南

### 1. 策略工厂

#### 1.1 查看策略总览

**步骤**：

1. 访问：http://47.103.19.238:8002/admin
2. 点击"策略总览"
3. 查看策略列表

**列表信息**：

| 列名 | 说明 |
|------|------|

**操作**：

- 详情：查看详情
- 发布：发布策略（草稿状态）
- 删除：删除策略

#### 1.2 新建策略

**步骤**：

1. 点击"策略总览"
2. 点击"新建策略"按钮
3. 完成 5 步向导

**步骤 1：基本信息**

| 字段 | 说明 | 必填 |
|------|------|------|
| 策略名称 | 如：燃油车续保策略 | ✅ |
| 策略描述 | 策略说明 | ❌ |
| 生效机构 | 选择适用地区 | ✅ |

**步骤 2：业务单元划分**

1. 选择业务单元（可多选）
2. 查看业务单元详情
3. 确认选择

**可用业务单元**：
- BU001：燃油车 - 续保
- BU002：燃油车 - 新保
- BU003：新能源 - 高价值
- BU004：新能源 - 普通
- BU005：豪车 - 续保

**步骤 3：成本模拟**

配置全局参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 固定成本率 | 10% | 运营成本占比 |
| 目标赔付率 | 75% | 期望赔付水平 |
| 市场费用率 | 12% | 渠道费用占比 |
| 自主系数下限 | 0.70 | 最低折扣 |
| 自主系数上限 | 1.30 | 最高折扣 |
| 反算折扣 | 是 | 是否反算 |

**步骤 4：收益模拟**

查看 5 种情景的收益预测：

| 情景 | 说明 |
|------|------|
| 情景 1 | 收紧核保 |
| 情景 2 | 轻微收紧 |
| 基准 | 当前策略 |
| 情景 4 | 轻微放宽 |
| 情景 5 | 放宽核保 |

**步骤 5：确认保存**

1. 确认策略信息
2. 点击"保存策略"
3. 策略状态变为"草稿"

### 2. 策略实验室

#### 2.1 模拟测试

**步骤**：

1. 点击"策略实验室"
2. 选择"模拟测试"
3. 选择测试策略

6. 等待测试完成（约 2 秒）
7. 查看测试结果

**测试设置**：

| 字段 | 说明 |
|------|------|
| 测试策略 | 选择要测试的策略 |
| 测试样本 | 1000/5000/10000 条 |
| 对比基准 | 当前线上策略/历史最优 |
| 适用机构 | 选择测试机构 |
| 业务单元 | 选择测试业务单元 |
| 测试描述 | 说明测试目的 |

**测试结果**：

| 指标 | 基准策略 | 测试策略 | 变化 |

| 成交率 | 46% | 54% | ↑17% |
| 利润 | 32 万 | 57 万 | ↑78% |

**结论**：
- ✅ 通过：各项指标提升
- ❌ 失败：指标下降

#### 2.2 A/B 测试

**步骤**：

1. 点击"策略实验室"
2. 选择"A/B 测试"
3. 设计实验方案
4. 设置实验组和对照组
5. 分配流量
6. 启动实验
7. 等待实验完成
8. 分析结果

**实验设计**：

| 配置项 | 说明 |
|--------|------|
| 实验名称 | 如：燃油车续保策略优化 |
| 实验假设 | 如：降低自主系数，成交率提升>15% |
| 对照组 | 当前线上策略 |
| 实验组 | 新策略 |
| 流量分配 | 50%/50% |
| 实验周期 | 7 天/14 天/30 天 |

**实验结果**：

| 指标 | 对照组 | 实验组 | 提升 |
|------|--------|--------|------|
| 成交率 | 45% | 52% | +15.6% |
| 利润率 | 3.2% | 2.8% | -12.5% |
| 保费规模 | 100 万 | 115 万 | +15% |

**结论**：
- ✅ 通过：主要指标提升
- ❌ 失败：主要指标下降

**自动调整规则**：

| 规则类型 | 阈值 | 调整系数 |
|---------|------|---------|
| 额度使用率 | > 80% | × 0.80 |
| 额度使用率 | > 100% | × 0.50 |
| 赔付率 | > 80% | × 0.85 |
