## 📐 1. 业务分群

### 什么是业务分群？

将整体业务按不同维度拆分为独立的业务单元（Business Unit），每个单元独立核算收入、成本、利润。

### 分群维度

| 维度 | 示例 | 说明 |
|------|------|------|
| **地区** | 上海、北京、广东 | 不同地区的风险特征不同 |
| **客户类型** | 新保、续保 | 新客获取成本 vs 老客维护成本 |
| **渠道类型** | 直销、代理、经纪 | 不同渠道的费用率不同 |
| **产品类型** | 燃油车、新能源车、豪车 | 不同产品的风险定价不同 |

### 业务单元示例

```
BU001: 燃油车 - 续保 - 直销 - 上海
  - 保单数：5000 单
  - 平均保费：3500 元/单
  - 纯风险成本：2520 元/单
  - 预期利润率：6.0%

BU002: 燃油车 - 新保 - 直销 - 上海
  - 保单数：2000 单
  - 平均保费：4800 元/单
  - 纯风险成本：3840 元/单
  - 预期利润率：-2.0%（战略亏损）
```

### 分群价值

- ✅ **精细化核算** - 清楚知道每个业务单元的盈亏
- ✅ **精准定价** - 不同业务单元采用不同定价策略
- ✅ **资源优化** - 将资源投入到 profitable 的业务单元
- ✅ **风险隔离** - 避免亏损业务拖累整体

## 🧮 2. UE 建模（单位经济模型）

### 什么是 UE 模型？

Unit Economics（单位经济模型）是分析单个业务单元经济性的方法，核心是量化每个单位的收入、成本和利润。

### 核心公式

```
收入 = 保单数 × 平均保费

成本 = 纯风险成本 + 固定成本 + 市场费用

利润 = 收入 - 成本

利润率 = 利润 / 收入
```

### 详细拆解

```
1. 收入端
   ├── 保单数（政策影响）
   └── 平均保费（定价影响）

2. 成本端
   ├── 纯风险成本（精算预设，元/单）
   │   = 平均保费 × 目标赔付率
   │
   ├── 固定成本（运营成本）
   │   = 收入 × 固定成本率
   │
   └── 市场费用（渠道费用）
       = 收入 × 市场费用率

3. 利润端
   └── 利润率 = 1 - 赔付率 - 市场费用率 - 固定成本率
```

### 示例计算

**业务单元：燃油车 - 续保 - 上海**

```
基础数据：
  - 保单数：5000 单
  - 平均保费：3500 元/单
  - 目标赔付率：72%
  - 市场费用率：12%
  - 固定成本率：10%

计算过程：
  保费规模 = 5000 × 3500 / 10000 = 1750 万元
  纯风险成本 = 3500 × 72% = 2520 元/单
  总纯风险成本 = 5000 × 2520 / 10000 = 1260 万元
  固定成本 = 1750 × 10% = 175 万元
  市场费用 = 1750 × 12% = 210 万元
  总成本 = 1260 + 175 + 210 = 1645 万元
  利润 = 1750 - 1645 = 105 万元
  利润率 = 105 / 1750 = 6.0%

快速计算：
  利润率 = 1 - 72% - 12% - 10% = 6.0% ✅
```

## 🔮 3. 情景仿真

### 什么是情景仿真？

通过设定不同的政策情景，模拟业务指标的变化，提前预估政策调整的影响。

### 5 种标准情景

| 情景 | 赔付率调整 | 保单数调整 | 业务含义 |
|------|-----------|-----------|---------|
| **情景 1** | -4% | -8% | 收紧核保 |
| **情景 2** | -2% | -4% | 轻微收紧 |
| **基准** | 0% | 0% | 当前策略 |
| **情景 4** | +2% | +4% | 轻微放宽 |
| **情景 5** | +4% | +8% | 放宽核保 |

### 为什么是这个比例？

**业务逻辑**：
- 赔付率下降（收紧核保）→ 部分客户被拒 → 保单数下降
- 赔付率上升（放宽核保）→ 更多客户准入 → 保单数上升
- 但保单数变化幅度 < 赔付率变化幅度（需求弹性）

### 情景计算示例

**基准情景**：
```
赔付率：72%
保单数：5000 单
利润率：6.0%
利润：105 万元
```

**情景 1（收紧核保）**：
```
赔付率：72% × (1 - 4%) = 69.12%
保单数：5000 × (1 - 8%) = 4600 单

新利润率 = 1 - 69.12% - 12% - 10% = 8.88%
新保费规模 = 4600 × 3500 / 10000 = 1610 万元
新利润 = 1610 × 8.88% = 143 万元

利润变化 = (143 - 105) / 105 = +36% ✅
```

**情景 5（放宽核保）**：
```
赔付率：72% × (1 + 4%) = 74.88%
保单数：5000 × (1 + 8%) = 5400 单

新利润率 = 1 - 74.88% - 12% - 10% = 3.12%
新保费规模 = 5400 × 3500 / 10000 = 1890 万元
新利润 = 1890 × 3.12% = 59 万元

利润变化 = (59 - 105) / 105 = -44% ❌
```

### 情景分析价值

- ✅ **提前预估** - 政策调整前知道影响
- ✅ **风险量化** - 用数据说话，避免拍脑袋
- ✅ **最优决策** - 找到利润最大化的策略点
- ✅ **倒 U 型曲线** - 验证当前策略是否最优

## 🎯 4. 策略优化

### 什么是策略优化？

基于情景仿真结果，找到风险与收益的最佳平衡点，实现利润最大化。

### 优化目标

```
Maximize: 利润 = 保费规模 × 利润率

Subject to:
  - 赔付率 ∈ [65%, 80%]  (风险可控)
  - 保单数变化 ∈ [-15%, +15%]  (业务稳定)
  - 利润率 ≥ 3%  (盈利要求)
```

### 优化方法

#### 方法 1：情景对比法

通过 5 种标准情景，找到利润最高的情景。

```
情景 1 利润：143 万
情景 2 利润：125 万
基准利润：105 万 ← 当前策略
情景 4 利润：82 万
情景 5 利润：59 万

结论：应该收紧核保（情景 1 最优）
```

#### 方法 2：边际分析法

计算赔付率每变化 1%，利润的变化量。

```
赔付率下降 1% → 利润率上升 1% → 利润增加 X 万
赔付率上升 1% → 利润率下降 1% → 利润减少 Y 万

当边际收益 = 边际成本时，达到最优
```

### 优化案例

**案例：新保业务是否应该收紧核保？**

**现状**：
```
业务单元：燃油车 - 新保 - 上海
当前赔付率：80%
当前利润率：-2.0%（亏损）
当前利润：-38 万元
```

**情景分析**：
```
情景 1（收紧）：赔付率 76.8%，利润 -15 万（减亏）
情景 2（微紧）：赔付率 78.4%，利润 -26 万（减亏）
基准：赔付率 80.0%，利润 -38 万（亏损）
情景 4（放宽）：赔付率 81.6%，利润 -51 万（增亏）
情景 5（大幅放宽）：赔付率 83.2%，利润 -64 万（增亏）

结论：应该收紧核保，情景 1 最优
```

**决策建议**：
- ✅ 收紧核保政策
- ✅ 提高准入门槛
- ✅ 虽然保单数下降 8%，但利润减亏 23 万

## 📊 业务指标体系

### 结果指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **保费规模** | 保单数 × 平均保费 | 业务规模 |
| **纯风险成本** | 保单数 × 纯风险成本（元/单） | 风险成本 |
| **利润率** | 1 - 赔付率 - 市场费用率 - 固定成本率 | 盈利能力 |
| **利润** | 保费规模 × 利润率 | 最终结果 |

### 过程指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **赔付率** | 纯风险成本 / 平均保费 | 风险水平 |
| **成交率** | 成交数 / 询价数 | 转化效率 |
| **保单数变化** | (新保单数 - 原保单数) / 原保单数 | 业务影响 |

### 健康度指标

| 指标 | 健康范围 | 说明 |
|------|---------|------|
| **利润率** | ≥ 5% | 盈利健康 |
| **赔付率** | 65% - 75% | 风险可控 |
| **保单数变化** | -10% ~ +10% | 业务稳定 |
