## 💰 金融科技

### 场景 1：信贷定价

**业务背景**：
银行/消金公司需要为不同客群制定差异化的贷款利率，平衡风险与收益。

**映射关系**：

| 车险概念 | 信贷概念 | 说明 |
|---------|---------|------|
| 保单数 | 放款笔数 | 业务规模 |
| 平均保费 | 单笔放款金额 | 客单价 |
| 赔付率 | 违约率/坏账率 | 风险成本 |
| 纯风险成本 | 预期损失（EL） | 风险定价基础 |
| 市场费用率 | 获客成本率 | 渠道费用 |
| 固定成本率 | 运营成本率 | 固定开支 |

**核心公式**：
```
利润率 = 利率 - 违约率 - 获客成本率 - 运营成本率
```

**情景仿真**：
```
情景 1：利率 +1%，违约率 -0.5%，放款数 -8%
情景 2：利率 +0.5%，违约率 -0.25%，放款数 -4%
基准：当前策略
情景 4：利率 -0.5%，违约率 +0.25%，放款数 +4%
情景 5：利率 -1%，违约率 +0.5%，放款数 +8%
```

**决策价值**：
- ✅ 找到最优利率点
- ✅ 量化风险调整后的收益
- ✅ 避免过度风险承担

## 💻 SaaS 服务

### 场景 1：订阅定价

**业务背景**：
SaaS 公司需要为不同套餐制定价格，平衡获客与收入。

**映射关系**：

| 车险概念 | SaaS 概念 | 说明 |
|---------|---------|------|
| 保单数 | 订阅用户数 | 业务规模 |
| 平均保费 | ARPU（单用户收入） | 客单价 |
| 赔付率 | 服务成本率 | 交付成本 |
| 市场费用率 | 营销费用率 | 获客成本 |
| 固定成本率 | 研发/管理费率 | 固定开支 |

**核心公式**：
```
利润率 = ARPU - 服务成本 - 获客成本 - 固定成本
```

**套餐设计示例**：

```
BU001: 基础版
  - 用户数：10000
  - ARPU: 99 元/月
  - 服务成本率：20%
  - 获客成本率：30%
  - 固定成本率：20%
  - 利润率：30%

BU002: 专业版
  - 用户数：5000
  - ARPU: 299 元/月
  - 服务成本率：25%
  - 获客成本率：35%
  - 固定成本率：20%
  - 利润率：20%

BU003: 企业版
  - 用户数：1000
  - ARPU: 999 元/月
  - 服务成本率：30%
  - 获客成本率：40%
  - 固定成本率：20%
  - 利润率：10%
```

**情景仿真**：
```
情景 1：专业版涨价 20%，用户数 -10%，利润率 +8%
情景 2：基础版降价 10%，用户数 +15%，利润率 -3%
...
```

**决策价值**：
- ✅ 优化套餐定价
- ✅ 平衡用户增长与收入
- ✅ 提升 LTV/CAC

## 🛡️ 保险科技

### 场景 1：精算定价

**业务背景**：
保险公司需要为不同险种制定费率，平衡竞争力与盈利。

**映射关系**：

| 车险概念 | 财险概念 | 说明 |
|---------|---------|------|
| 保单数 | 承保件数 | 业务规模 |
| 平均保费 | 件均保费 | 客单价 |
| 赔付率 | 综合赔付率 | 风险成本 |
| 市场费用率 | 渠道费用率 | 佣金/手续费 |
| 固定成本率 | 管理费率 | 固定开支 |

**核心公式**：
```
综合成本率 = 赔付率 + 费用率 + 管理费率
利润率 = 1 - 综合成本率
```

**情景仿真**：
```
情景 1：费率 +5%，赔付率 -3%，件数 -8%
情景 2：费率 +3%，赔付率 -2%，件数 -5%
基准：当前策略
情景 4：费率 -3%，赔付率 +2%，件数 +5%
情景 5：费率 -5%，赔付率 +3%，件数 +8%
```

**决策价值**：
- ✅ 找到最优费率点
- ✅ 量化价格弹性
- ✅ 平衡规模与利润

## 📊 跨行业通用指标

### 结果指标

| 指标 | 通用公式 | 说明 |
|------|---------|------|
| **业务规模** | 订单数 × 客单价 | 收入规模 |
| **可变成本** | 订单数 × 单均成本 | 直接成本 |
| **利润率** | 1 - 可变成本率 - 固定成本率 | 盈利能力 |
| **利润** | 业务规模 × 利润率 | 最终结果 |

### 过程指标

| 指标 | 通用公式 | 说明 |
|------|---------|------|
| **成本率** | 成本 / 收入 | 成本效率 |
| **转化率** | 转化数 / 曝光数 | 转化效率 |
| **订单数变化** | (新订单数 - 原订单数) / 原订单数 | 业务影响 |

### 健康度指标

| 指标 | 健康范围 | 说明 |
|------|---------|------|
| **利润率** | ≥ 5% | 盈利健康 |
| **成本率** | ≤ 70% | 成本可控 |
| **订单数变化** | -10% ~ +10% | 业务稳定 |
